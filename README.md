##Create page 
````ionic
ionic g page myPage

√ Create app/pages/my-page/my-page.html
√ Create app/pages/my-page/my-page.ts
√ Create app/pages/my-page/my-page.scss
````

##Create a service
````ionic
ionic g provider MyData

√ Create app/providers/my-data/my-data.ts
````
